Project Overview

Dappmine is a minimalistic web application designed for mining cryptocurrencies which are CPU compatible and ASIC resistant. It utilizes modified versions of the CryptoNight algorithms (CryptoNight & CryptoNight V7) embedded with asynchronous javascript functionality for a profitable mining experience. The Dappmine codebase currently targets MINTME.com & Dero but will included added support for other webchain tokens such as Bytecoin & Lonze.
